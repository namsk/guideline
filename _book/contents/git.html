Git 소개

여러가지 소스코드관리(SCM, Source Code Management) 툴이 있지만 레일스에서 일반적으로 Git을 주로 사용한다.

무료 온라인 문서

Git에 처음 접하는 경우에는 Pro Git 온라인 문서를 참고하면 도움이 된다.

설치하기

운영체제별 git 설치방법은 Pro Git에 잘 설명되어 있다. 맥 사용자들은 Mac OS X에서 git의 한글호환 오류문제가 있을 수 있다는 것을 알아 둘 필요가 있으며, 더 자세한 내용은 여기를 참고하기 바란다.
